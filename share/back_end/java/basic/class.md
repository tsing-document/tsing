# 面向对象
- 概念：
    - 1.类：
        - 概念：
            - 类相当于一个模版、相当于一个模具。
        - 类之间的关系：
            - 依赖：
                - user-a
            - 聚合：
                - has-a
            - 继承：
                - is-a
        - 注意：
            - 构造器和类同名
            - 每个类可以有一个以上的构造器
            - 构造器可以有 0个、1个或者多个参数
            - 构造器没有返回值
            - 构造器总是伴随这个new操作一起调用
    - 2.对象：
        - 概念：
            - 对象的三个特征：
                - 对象的行为
