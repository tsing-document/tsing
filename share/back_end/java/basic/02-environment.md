#### 环境搭建

- 下载 JDK并设置环境变量：
    - 略

- Hello World：
    - 1.编写类 Demo.java：
        ```java
            public class Demo {
                public static void main (String[] args) {
                    String greeting = "Say Hello World!";
                    System.out.println(greeting);
                }
            }
        ```

    - 2.编译类：
        ```bash
            javac Demo.java
        ```

    - 3.执行：
        ```bash
            java Demo
        ```
