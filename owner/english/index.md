# 英语
- # 一、句子成分与基本句型
    - 定义：
        - 句子成分包括：
            - 主语
            - 谓语
            - 宾语
            - 表语
            - 补语
            - 定语
            - 状语
            - 还有一个不重要的是同位语
        - 分析：
            - 主语：
                - 一句话主要叙述的对象或者这句话主要讲的内容。主语在英语的肯定句中，一般就是放开头。
                    - 充当主语的词语有：
                        - 名词：
                            - 名词都可以作为主语。名词就是世间万物的名称。人名、地名、普通事物的名称。人名、地名一般都称为 `专有名词`，第一个字母任何时候都是大写。`普通名词` 案例：computer、teacher。。。，任何一个普通名词都可以做主语，但是做主语的时候有点要求，规则：`普通名词` 分为可数名词和不可数名词。
